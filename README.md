## Setup
To run any project follow the procedure:

- Set the script executable permission by running chmod command:

`chmod +x file_name.sh`

- Run or execute the script using the following syntax:

`./file_name.sh`

or

`sh file_name.sh`

or

`bash file_name.sh`
